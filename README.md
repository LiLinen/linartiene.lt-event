# saves-pazinimo-grupe.lt

A promotional static web site for [saves-pazinimo-grupe.lt](https://saves-pazinimo-grupe.lt).

## Installation
```bash
$ npm install
```

## Build
```bash
$ npm run build
```

Production files will be generated in the `dist` directory.
